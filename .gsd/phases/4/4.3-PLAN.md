# Plan 4.3: Dashboard UI тАФ Summary & Detail Views

## Objective

Build the actual dashboard UI inside the sidebar with two views: a Summary view (list of tracked emails with open/click counts) and a Detail view (per-recipient activity timeline for a single email). Wire both views to real-time Firestore data.

## Context

- src/api/db.js тАФ Firestore query functions from Plan 4.1 (subscribeToEmails, subscribeToEvents)
- src/dashboard/DashboardApp.jsx тАФ Placeholder from Plan 4.2
- src/content.js тАФ Auth flow (getAuthToken, authenticateFirebase)
- .gsd/DECISIONS.md тАФ Summary + Detail views, real-time updates, aggregate stats nice-to-have

## Data Shape Reference

```
Email doc:   { userId, subject, recipients: [{email, id}], sentAt }
Event doc:   { type: "open"|"click", recipientId, timestamp, userAgent, ip, targetUrl? }
```
